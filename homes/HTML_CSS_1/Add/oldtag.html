Коротко о некоторых HTML тегах

Тег <code>
Данный тег используется для вывода фрагментов программного кода

Например, ваш код
ваша программа
<code> фрагмент программного кода </code>
продолжение вашей программы

Будет выглядить следующим образом
ваша программа
фрагмент программного кода
продолжение вашей программы

Тег <tt>
Данный тег отображает текст моноширинным шрифтом

Например, ваш код:
<tt>Привет, как дела? </tt>

Будет выглядеть так:
Привет, как дела?

Тег <kbd>
Данный тег используется для обозначения текста, который набирается на клавиатуре или для названия клавиш

Например, ваш код:
Для вставки текста нажмите <kbd>Ctrl V</kbd>

Будет выглядеть так:
Для вставки текста нажмите Ctrl V

Теги <sub> и <sup>
Данные теги используются для отображения верхних и нижних символов, соответсвенно

Например, ваш код:
Формула воды H<sub>2</sub>O, а формула Эйнштейна E=m*c<sup>2</sup>

Будет выглядеть так:
Формула воды H2O, а формула Эйнштейна E=m*c2

Теги <ins> и <del>
Данные теги удобны при формировании списков для отображения запланированных (обозначаются подчеркиванием в списке) и выполненных задач (обозначаются перечеркиванием)

Например, ваш код:

<strong>Список дел:</strong>
<ul>
<li>Помыть посуду</li>
<li>Сделать уроки</li>
<li>Полить цветы</li>
<li><ins>Погулять с собакой</ins> </li>
<li><del>Пропылесосить комнату</del> </li>
</ul>

Будет выглядеть так:

Список дел:

  • Помыть посуду
  • Сделать уроки
  • Полить цветы
  • Погулять с собакой
  • Пропылесосить комнату

Тег <q>
Данный тег предназначен для выделения цитат внутри предложения

Например, ваш код:
Евгений Леонов говорил:<q>Обиды вообще не следует копить. Не большое, как говорится, богатство</q>

Будет выглядеть так:
Евгений Леонов говорил: Обиды вообще не следует копить. Не большое, как говорится, богатство

Тег <nobr>
Данный тег уведомляет браузер отображать текст без переносов

Например, ваш код:
<nobr>Lorem ipsum dolor, sit amet consectetur adipisicing elit. Commodi illum doloribus porro esse at, laborum obcaecati ab dolor sapiente quisquam et maiores dolorem ipsam! Praesentium quos dicta explicabo sapiente quam.<nobr>

Будет выглядеть без переносов так:
Lorem ipsum dolor, sit amet consectetur adipisicing elit. Commodi illum doloribus porro esse at, laborum obcaecati ab dolor sapiente quisquam et maiores dolorem ipsam! Praesentium quos dicta explicabo sapiente quam.

Тег <wbr>
Данный тег указывает браузеру место, где допускается делать перенос строки в тексте, если этого требует ширина родительского элемента

Например, очень длинное слово:
превысоко<wbr>многорассмотри<wbr>тельствующий

Будет выглядеть так:
превысокомногорассмотрительствующий
