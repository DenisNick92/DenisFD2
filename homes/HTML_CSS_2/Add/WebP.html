Немного о формате изображений WebP

Достоинства: WebP имеет ряд преимуществ по сравнению с другими форматами изображений. Он позволяет уменьшить размер файлов изображений на 25-35% по сравнению с JPEG, сохраняя при этом качество изображения. WebP обеспечивает поддержку прозрачности, анимации и многоканальности.

Соотношение между размерами форматов webp, jpeg и png составляет примерно 5:6:50. Поэтому на ресурсах с большим количеством графических изображений разница будет ощутима.

Недостатки: Safari и Internet Explorer не считывают этот формат.
